# Snake Game

A classic Snake game implementation in Java using Swing.

## Features
- Snake movement with arrow keys
- Food spawning randomly
- Collision detection (walls and self-collision)
- Score tracking
- Game over screen with restart functionality
- Smooth gameplay
- Prevents 180-degree turns

## How to Run
1. Compile both Java files: `javac SnakeGame.java GamePanel.java`
2. Run the game: `java SnakeGame`

## Controls
- **Arrow Keys** - Move the snake (Up, Down, Left, Right)
- **Space** - Restart the game when game over

## Game Rules
- Eat the red food to grow longer and increase your score
- Don't hit the walls
- Don't hit yourself
- Each food eaten gives you 10 points

Enjoy the game!
